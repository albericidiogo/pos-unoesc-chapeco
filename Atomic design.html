Atomic Design

Criado pelo web design Brad Frost, é um tipo modular que trata paginas na internet como sistemas, de maneira semelhante a átomos, moléculas e organismos.

Átomos

atomos são as tags, elementos soltos como inputs, labels, fomrs e campos de formulario. eles sâo a base para construir elementos maiores.

Moléculas

são os agrupamentos de atomos, elas fazem com que componentes isolados funcionem com um propósito.

Organismo

são um conjunto de de moléculas, correspondem a uma seção do site tais como; footer, header e sidebar. podem ter diversos propositos paralelos. com esta metodologia podemos padronizar a criação de sistemas de design com partes reutilizaveis. um ponto importante e que podemos fazer ajuste e mudancas facilmente alterando a ordem ou a combinação dos elementos para criar novas paginas.
